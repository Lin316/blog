Vuex 概述

什么是 Vuex

Vuex 是一个 Vue 插件，用于管理 Vue 应用中多组件共享的数据。

使用场景：

  • 某个状态在多个组件中使用（如：用户个人信息）
  • 多个组件共同维护一份数据（如：购物车数据）

Vuex 的优点：

  • 数据集中化管理：所有共享数据统一存储
  • 响应式变化：数据变化自动更新视图
  • 操作简洁：提供辅助函数简化操作
  • 数据流清晰：单向数据流便于追踪和调试

不是所有场景都适合使用 Vuex，使用 Vuex 会增加项目复杂度（引入更多概念），小型项目使用 $emit / $on 或事件总线即可。

Vuex 的核心概念

Vuex 包含以下核心概念：

概念 说明 用途
State 状态（数据） 存储共享数据
Mutations mutations 同步修改 state
Actions 动作 处理异步操作，提交 mutations
Getters 计算属性 从 state 派生数据（筛选、计算）
Modules 模块 模块化组织 Vuex（大型项目）
