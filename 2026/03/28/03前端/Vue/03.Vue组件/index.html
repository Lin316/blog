非父子通信

事件总线 Event Bus

Event Bus（事件总线）是一种基于发布-订阅模式的跨组件通信机制。它通过一个全局的 Vue 实例作为事件中心，实现任意组件之间的解耦通信。

核心原理：

  • 创建一个全局的 Vue 实例作为事件中心
  • 组件 A 通过 $emit 触发事件并传递数据
  • 组件 B 通过 $on 监听事件并接收数据
  • 组件销毁时需要手动移除监听器，防止内存泄漏

必须在组件销毁前调用 $off 或 off 移除监听器

跨层级通信 Provide/Inject

Provide/Inject 是 Vue 提供的依赖注入机制，允许祖先组件向其所有后代组件（无论层级多深）注入数据或方法，实现跨级组件通信。

核心特点：

  • 单向数据流：数据从祖先组件流向后代组件
  • 响应式更新：注入的对象是响应式的，数据变化会自动更新。

如果祖先注入的不是对象，是基本数据类型，是不会响应式更新的。
