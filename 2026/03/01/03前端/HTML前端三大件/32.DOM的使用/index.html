DOM 介绍

DOM 树

DOM（Document Object Model）：浏览器把 HTML 文档解析成一棵“对象树”，树上的每个节点都是一个对象。

DOM 节点

  • 根节点：html 标签，也就是document.documentElement
  • 元素节点：div、a、body……
  • 文本节点：标签之间的文字
  • 属性节点：是指 HTML 标签中的属性，如上图中 a 标签的 href 属性、div 标签的 class 属性。

查询 DOM 节点

查询选择器

  • document.querySelector(选择器)：返回第一个匹配元素，找不到返回 null
  • document.querySelectorAll(选择器)：返回所有匹配元素的 NodeList（可遍历，但不是数组）

查询父节点

  • node.parentNode：返回当前节点的父节点
  • node.parentElement：返回当前节点的父元素节点

查询子节点

  • node.children：返回当前元素的所有子元素节点，不包含文本节点
  • node.firstElementChild：返回第一个子元素
  • node.lastElementChild：返回最后一个子元素

查询兄弟节点

  • node.nextElementSibling：返回下一个兄弟元素。
  • node.previousElementSibling：返回上一个兄弟元素。

修改元素内容

innerHTML

获取或设置元素的 HTML 结构

特点：

  • 返回完整 HTML 字符串（包含标签）
  • 不关心 CSS 是否隐藏，不会做“可见性判断”

示例：

innerText

获取或设置元素中 可见的文本内容

特点：

  • 只返回“渲染后”的文本
  • 会忽略 HTML 标签
  • 会受 CSS 影响（如 display: none 不会被读取）
  • 会触发浏览器重排（性能略低）

示例：

textContent

获取或设置元素中 文本内容（包括不可见的文本内容）

特点：

  • 会忽略 HTML 标签
  • 不受 CSS 影响
  • 不触发重排
  • 性能最好

修改元素属性

通过属性名修改

表单中常用的属性：

  • input.value：输入值
  • input.type：输入类型
  • checkbox.checked：是否勾选
  • select.value / select.selectedIndex：选中项
  • button.disabled：是否禁用

修改元素样式

通过 style 对象修改

先通过元素 DOM 对象，获取该元素 DOM 对象的 style 样式对象，再通过 style 样式对象修改样式。

通过 className 修改

通过修改元素的 class 属性，从而修改元素样式。

注：使用 className 修改元素的 class 属性，会覆盖原来的 class 属性值。

通过 classList 修改

使用 classList 方式对元素的 class 属性追加或删除类名

classList 方法 作用
add() 添加 class
remove() 删除 class
toggle() 有则删，无则加
contains() 判断是否存在

修改自定义属性值

在标签中，自定义的属性名必须以 data- 开头，然后通过 DOM对象 中的 dataset对象 获取自定义属性值。
