模块介绍

path 模块是一个用于处理文件路径的核心模块。

引入模块：

主要作用：

  1. 处理文件路径
  2. 规范化路径
  3. 拼接路径
  4. 获取文件名、扩展名、目录
  5. 解析路径结构

特点：

  • 自动处理 Windows / Linux 路径差异
  • 提供 同步工具函数
  • 不访问文件系统（只处理字符串）
