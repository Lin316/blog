Node.js 介绍

Node.js 是一个 JavaScript 运行环境，可以让 JavaScript 在 浏览器之外（服务器端）运行。

Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境，让 JavaScript 既能写前端，又能写后端。

CommonJS 模块

CommonJS 模块介绍

CommonJS 是 Node.js 最早的模块规范。

核心 API：

  • require()：加载模块
  • module.exports：导出模块
  • exports：导出模块

require()

用于 加载模块，加载自己写的文件模块，模块路径必须用 ./ 或 ../

module.exports

用于 导出模块

exports

exports 是 module.exports 的一个引用，exports === module.exports 为 true。

不要在同一个模块中同时使用 exports 和 module.exports

模块缓存机制

Node.js 模块 只会加载一次。

多次 require 模块也只会加载一次，加载后的模块会缓存在 require.cache 中。

npm 包管理

npm 介绍

npm（Node Package Manager） 是 Node.js 自带的包管理工具。

主要作用：

  • 安装第三方包
  • 管理项目依赖
  • 运行脚本
  • 管理版本

包分类

项目包

项目包是只在当前项目中使用的依赖包。

安装后会记录在项目的 package.json 中，并放在项目目录下：

特点 说明
作用范围 当前项目
安装位置 project/node_modules
是否记录 记录在 package.json
使用方式 require / import
常见用途 项目依赖库

全局包

全局包是安装到系统环境中的工具包，通常是 命令行工具。

安装后可以在 任何目录 使用。

特点 说明
作用范围 整个系统
安装位置 全局 node_modules
是否记录 不记录在项目 package.json
使用方式 命令行
常见用途 CLI工具
