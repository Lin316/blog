对象（Object）是一种引用数据类型，用于存储 键值对（key-value） 数据。

  • name、age、isStudent → 属性（property）
  • "Tom"、18、true → 属性值

对象原型

每一个函数都会有一个 prototype 属性，这个属性指向一个对象，这个对象就是原型对象。

这个原型对象的作用是：

让所有实例共享属性和方法

原型链

JS 查找属性的规则：先找自己 → 再找原型 → 再找原型的原型 → 一直到 null

继承实现

原型继承

缺点：父类的属性被子类共享，子类对象修改父类的对象属性，其它子类对象也会受影响。

构造函数继承

构造函数继承：继承父类构造函数中涉及的属性和方法。

优点：每个子类都会独有一份父类属性，父类属性不被子类共享。

缺点：不能继承父类的 prototype 原型对象中的内容。

组合继承

组合继承：原型继承 + 构造函数继承，取两种继承的优点。

缺点：要调用两次父类的构造函数。

寄生组合继承

由 Object.create() 创建一个空对象，并指定这个空对象的原型对象为父类的原型对象。
