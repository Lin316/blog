Axios 介绍

Axios 主要用于在浏览器和 Node.js 中发送 HTTP 请求。

Axios 常用方法

Axios 函数 说明
axios(config) 通用/最本质的发任意类型请求的方式
axios.get(url[, config]) 发 get 请求
axios.delete(url[, config]) 发 delete 请求
axios.post(url[, data[, config]]) 发 post 请求
axios.put(url[, data[, config]]) 发 put 请求
axios.create(config) 创建一个新的 axios 对象
axios.defaults.xxx axios 默认全局 config 配置
axios.all(promises) 用于批量执行多个异步请求

Axios 支持的配置

Axios 支持配置参数：

配置 说明
method 请求方式
url 请求地址
headers 请求头
params GET参数
data POST参数
timeout 超时时间
responseType 返回数据类型
baseURL 基础 URL
signal 由 AbortController 对象控制请求是否中断
