Vue 指令介绍

Vue 指令：带有 v- 前缀的特殊属性，用于操作 DOM。

作用：在 HTML 上添加特殊属性，让 Vue 根据数据自动更新页面。

分类 指令 作用
属性绑定 v-bind 把数据绑定到标签属性上
事件绑定 v-on 监听 DOM 事件并执行方法
双向绑定 v-model 表单值和数据保持同步
条件渲染 v-show / v-if / v-else-if / v-else 按条件显示或隐藏内容
列表渲染 v-for 根据数组或对象循环生成结构
内容渲染 v-text / v-html 渲染纯文本或 HTML 内容
