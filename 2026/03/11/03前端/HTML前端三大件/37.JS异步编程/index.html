Promise 的介绍

Promise 表示一个 未来才会完成的操作。

一个“承诺”：未来会返回 成功结果 或 失败原因。

Promise 有 三种状态：

状态 含义
pending 进行中
fulfilled 已成功
rejected 已失败

Promise 的使用

创建 Promise 对象

通过创建 Promise 对象进行异步操作：

如果异步代码正常执行，使用 resolve 返回执行结果。

如果异步代码执行异常，使用 reject 返回异常信息。

处理成功结果

使用 then() 方法接收异步代码的执行结果。

处理异常结果

使用 catch() 方法接收异步代码的异常信息。

async / await

async

async 用来标记函数，使得函数的返回值被封装为 Promise.resolve() 或 Promise.reject()。

被 async 修饰的函数，return 返回值被封装在 Promise.resolve() 中，throw 返回值被封装在 Promise.reject() 中

await

获取正常执行结果

await 用来等待 Promise 执行完成，并把 Promise.resolve() 结果赋值给变量。

await 必须在 async 修饰的函数中使用，否则语法错误。

就是把 then 方法中的入参值赋值给了 result 变量。

获取异常信息

如果异步代码返回 Promise.reject() ，那么使用 await 时候就需要使用 try catch 捕获 Promise.reject() 信息。

result 输出：时间戳是偶数，执行成功，时间戳：1773242523164。

err 输出：时间戳是奇数，执行失败，时间戳：1773242525949。

总结

await 会阻塞当前 async 函数，不会阻塞整个 JS 线程。

JS 是单线程，代码会一直执行下去，除非代码执行完成或遇到 Promise 异步代码，才会去执行另一块代码，另一块代码同样会一直执行下去，除非代码执行完成或遇到 Promise 异步代码。
