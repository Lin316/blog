Promise иЎЁзӨәдёҖдёӘ жңӘжқҘжүҚдјҡе®ҢжҲҗзҡ„ж“ҚдҪңгҖӮ

дёҖдёӘвҖңжүҝиҜәвҖқпјҡжңӘжқҘдјҡиҝ”еӣһ жҲҗеҠҹз»“жһң жҲ– еӨұиҙҘеҺҹеӣ гҖӮ

Promise жңү дёүз§ҚзҠ¶жҖҒпјҡ

зҠ¶жҖҒ еҗ«д№ү
pending иҝӣиЎҢдёӯ

rejected е·ІеӨұиҙҘ

еҰӮжһңејӮжӯҘд»Јз Ғжү§иЎҢејӮеёёпјҢдҪҝз”Ё reject иҝ”еӣһејӮеёёдҝЎжҒҜгҖӮ

async / await

async

async з”ЁжқҘж Үи®°еҮҪж•°пјҢдҪҝеҫ—еҮҪж•°зҡ„иҝ”еӣһеҖјиў«е°ҒиЈ…дёә Promise.resolve() жҲ– Promise.reject()гҖӮ

иў« async дҝ®йҘ°зҡ„еҮҪж•°пјҢreturn иҝ”еӣһеҖјиў«е°ҒиЈ…еңЁ Promise.resolve() дёӯпјҢthrow иҝ”еӣһеҖјиў«е°ҒиЈ…еңЁ Promise.reject() дёӯ

await

иҺ·еҸ–жӯЈеёёжү§иЎҢз»“жһң

await з”ЁжқҘзӯүеҫ… Promise жү§иЎҢе®ҢжҲҗпјҢе№¶жҠҠ Promise.resolve() з»“жһңиөӢеҖјз»ҷеҸҳйҮҸгҖӮ

await еҝ…йЎ»еңЁ async дҝ®йҘ°зҡ„еҮҪж•°дёӯдҪҝз”ЁпјҢеҗҰеҲҷиҜӯжі•й”ҷиҜҜгҖӮ

е°ұжҳҜжҠҠ then ж–№жі•дёӯзҡ„е…ҘеҸӮеҖјиөӢеҖјз»ҷдәҶ result еҸҳйҮҸгҖӮ

иҺ·еҸ–ејӮеёёдҝЎжҒҜ

еҰӮжһңејӮжӯҘд»Јз Ғиҝ”еӣһ Promise.reject() пјҢйӮЈд№ҲдҪҝз”Ё await ж—¶еҖҷе°ұйңҖиҰҒдҪҝз”Ё try catch жҚ•иҺ· Promise.reject() дҝЎжҒҜгҖӮ

result иҫ“еҮәпјҡж—¶й—ҙжҲіжҳҜеҒ¶ж•°пјҢжү§иЎҢжҲҗеҠҹпјҢж—¶й—ҙжҲіпјҡ1773242523164гҖӮ
