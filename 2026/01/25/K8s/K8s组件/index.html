Namespace 组件

Namespace 组件介绍

  1. 资源隔离：如测试环境下的资源，开发环境下的资源。
  2. 权限隔离：测试环境的 Namespace 只给测试用，开发环境 Namespace 只给开发用。
  3. 资源配额：可以为不同的 Namespace 分配硬件资源，如 CPU、内存等。

Pod 组件

Pod 组件介绍

程序运行在容器中，容器运行在 Pod 中，k8s 通过管理 Pod 进而管理程序。

一个 Pod 里面可以运行一个或多个容器。

Pod 的 yaml 配置文件

使用 -o wide 查看 Pod 信息

列名称 描述
NAME pod 的名称。
READY 描述 pod 里面有多少个容器，有多少个容器正在运行中。
STATUS 描述 pod 的状态，Running 指运行中，Terminating 指正在删除 pod。
RESTARTS pod 中容器的重启次数。
AGE pod 运行了多少时间。
IP k8s 给 pod 分配的 ip。
NODE pod 运行在集群哪个节点上。

使用 describe 查看详细信息

Deployment 组件

Deployment 组件介绍

Deployment 是 Pod 控制器，用来管理 Pod 的生命周期。Deployment 能够在 Pod 出现故障时，自动重启 Pod 或创建 Pod。

Deployment 的 yaml 配置文件

使用 -o wide 查看 Deployment 信息

列名称 描述
NAME deployment 的名称。
READY 描述 deployment 里面有多少个 pod，有多少个 pod 正在运行中。
UP-TO-DATE 有多少 pod 使用的是当前 deployment 最新配置。
AVAILABLE 可以对外提供服务的 pod 数。
AGE deployment 运行了多少时间。
CONTAINERS pod 中容器的名字。
IMAGES 容器使用的镜像。
SELECTOR 标签选择器，通过标签选择器来确定控制哪些 pod。

使用 describe 查看详细信息

Service 组件

使用 Service 组件，使 Pod 能够对外提供访问接口。Service 组件也能够实现多个 Pod 间的负载均衡。

Service 的 yaml 配置文件

使用 -o wide 查看 Service 信息

列名称 描述
NAME service 的名称。
TYPE 指定 service 的类型，NodePort：对外提供访问接口、ClusterIP：只对 k8s 集群内部提供访问接口。
CLUSTER-IP k8s 给 service 分配的 ip。
EXTERNAL-IP 无。
PORT(S) 左边 80 是 k8s 给 service 分配的端口，右边是 31595 是 service 对外提供访问的端口。
AGE service 运行了多少时间。
SELECTOR 标签选择器，通过标签选择器来确定对接哪些 pod。

使用 service 查看详细信息
