盒子模型属性

内容区域尺寸 width / height

控制元素内容区域的宽度和高度。

内边距 padding

控制内容到边框之间的间距。

简写规则：

  • padding: 上 右 下 左;
  • padding: 10px 20px; 表示 上下 10px，左右 20px
  • padding: 10px; 四个方向相同

特点：

  • 会增大盒子的实际尺寸
  • 背景会覆盖到 padding 区域

外边距 margin

控制盒子与其他盒子之间的外部间距。

常见属性：

  • margin
  • margin-top
  • margin-right
  • margin-bottom
  • margin-left

特点：

  • 不影响背景颜色，始终是透明区域
  • 垂直方向的 margin 可能发生外边距合并

当 左右 margin 设置为 auto 时：

  • 浏览器会自动把剩余空间平均分配给左右两边
  • 元素就会在父容器中水平居中

常见取值：

  • border-radius: 10px 20px; 水平、垂直方向不同

盒子阴影 box-shadow

用于给盒子添加投影效果。

基本语法：

说明：

  • x-offset：阴影在水平方向的偏移量
  • y-offset：阴影在垂直方向的偏移量
  • blur：模糊程度（越大阴影效果越柔和）
  • spread：扩展半径（控制阴影的大小，可省略）
  • color：阴影颜色

常见效果：

  • box-shadow: 0 0 10px rgba(0, 0, 0, 0.3);
  • box-shadow: 0 2px 8px rgba(0, 0, 0, 0.15);

背景图属性

背景颜色 background-color

用于设置元素的背景颜色。

常见写法：

  • background-color: red;
  • background-color: #ff0000;
  • background-color: rgb(255, 0, 0);
  • background-color: rgba(255, 0, 0, 0.5);

说明：

  • rgba 可以设置透明度
  • 默认值是 transparent（透明）

背景图片 background-image

用于设置背景图片。

说明：

  • 默认会平铺（重复）
  • 如果图片不存在，背景为空

背景平铺方式 background-repeat

控制背景图片是否重复。

常见取值：

值 说明
repeat 默认，水平垂直都重复
repeat-x 仅水平方向重复
repeat-y 仅读取垂直方向重复
no-repeat 不重复

背景位置 background-position

控制背景图片的位置。

属性值：水平方向位置 垂直方向位置

取值 位置
left 左侧
right 右侧
center 居中
top 顶部
bottom 底部

常见写法：

  • background-position: left top;
  • background-position: 100px 50px;
  • background-position: 50% 50%;

坐标

  • 水平：正数向右；负数向左
  • 垂直：正数向下；负数向上

背景大小 background-size

控制背景图片的尺寸。

常见取值：

值 说明
auto 默认大小
cover 覆盖整个元素，可能裁剪图片
contain 完整显示图片，可能在元素内留下空白
100px 50px 指定宽高（第一个是宽度，第二个是高度）

背景固定方式 background-attachment

控制背景是否随滚动条滚动。

常见取值：

值 说明
scroll 默认，随页面一起滚动
fixed 固定不动（视差效果）
local 随元素内容区域滚动

背景裁剪区域 background-clip

控制背景绘制区域。

常见取值：

值 说明
border-box 默认，从边框开始绘制
padding-box 从内边距区域开始绘制
content-box 只在内容区域内绘制

背景原点 background-origin

控制背景图片定位的起点。

常见取值：

值 说明
border-box 以边框区域作为定位参考
padding-box 以内边距区域作为定位参考
content-box 以内容区域作为定位参考

背景复合写法 background

可以把多个背景相关属性写在一起。

推荐书写顺序：

  • background: 颜色 图片 平铺 位置 / 大小 固定方式;

注意：

  • background-size 必须写在 / 后面
  • 其他属性顺序可以变化，但推荐保持规范写法

显示模式 display

显示模式 display 取值 说明
block 块级元素
inline 行内元素
inline-block 行内块元素

块级元素（block）

主要特点：

  • 独占一行
  • 可以设置 width、height
  • 可以设置 margin、padding
  • 元素从上到下纵向排列
  • 默认宽度等于父元素内容区域的 100%

行内元素（inline）

主要特点：

  • 不独占一行，多个行内元素在一行内水平排列
  • 宽高由内容决定
  • 不能设置 width、height
  • 左右 margin、padding 有效
  • 上下 margin 基本无效
  • 行内标签会被当做文字处理

注意：行内元素通常不直接设置宽高。

行内块元素（inline-block）

主要特点：

  • 不独占一行，可以和其他元素横向排列
  • 可以设置 width、height
  • 可以设置 margin、padding
  • 兼具行内元素和块级元素的特性
  • 行内块标签会被当做文字处理

三种显示模式对比

特性 block inline inline-block
是否独占一行 是 否 否
是否可设置宽高 是 否 是
上下 margin 支持 不支持 支持
左右 margin 支持 支持 支持
排列方式 纵向 横向 横向

Flex 布局属性

开启 Flex 布局 display

行内元素也可以开启 Flex：

设为 Flex 后：

  • 该元素称为容器（flex container）
  • 容器的直接子元素称为项目（flex item）
  • 项目默认沿主轴（main axis）排列，另一条轴为交叉轴（cross axis）

容器常用属性

主轴方向 flex-direction

控制主轴方向（项目的排列方向）。

常见取值：

  • row：主轴为水平方向，起点在左侧（默认）
  • row-reverse：水平方向，起点在右侧
  • column：垂直方向，起点在上方
  • column-reverse：垂直方向，起点在下方

是否换行 flex-wrap

默认所有项目都在一行内，flex-wrap 控制是否换行。

常见取值：

  • nowrap：不换行（默认）
  • wrap：换行，第一行在上方
  • wrap-reverse：换行，第一行在下方

复合写法 flex-flow

flex-flow 是 flex-direction 和 flex-wrap 的简写。

语法：

  • flex-flow: <flex-direction> <flex-wrap>;

主轴对齐方式 justify-content

控制项目在主轴上的对齐方式（假设主轴为从左到右）。

常见取值：

  • flex-start：靠主轴起点对齐（默认）
  • flex-end：靠主轴终点对齐
  • center：居中对齐
  • space-between：两端对齐，项目间距相等
  • space-around：项目两侧间距相等，项目间距是两端间距的两倍

交叉轴对齐方式 align-items

控制项目在交叉轴上的对齐方式（假设交叉轴从上到下）。

常见取值：

  • flex-start：交叉轴起点对齐
  • flex-end：交叉轴终点对齐
  • center：交叉轴中点对齐
  • baseline：项目的第一行文字的基线对齐
  • stretch：项目未设置高度或设为 auto 时，拉伸填满容器（默认）

多行交叉轴对齐 align-content

当存在多行（flex-wrap 生效）时，控制多行整体在交叉轴上的对齐方式；只有一行时无效。

常见取值：

  • flex-start：交叉轴起点对齐
  • flex-end：交叉轴终点对齐
  • center：交叉轴中点对齐
  • space-between：两端对齐，行间距平均
  • space-around：项目两侧间距相等，行间距是两端间距的两倍
  • stretch：行占满整个交叉轴

项目常用属性

排序 order

控制项目在主轴中的排列顺序，数值越小越靠前。

默认值为 0。

放大比例 flex-grow

控制项目在主轴剩余空间中的放大比例。

说明：

  • 默认值为 0，表示即使有剩余空间也不放大
  • 所有项目 flex-grow: 1 时，按比例平分剩余空间
  • 某个项目为 2，其他为 1 时，该项目占用的剩余空间是其他项目的两倍

缩小比例 flex-shrink

控制主轴空间不足时项目的缩小比例。

说明：

  • 默认值为 1，表示空间不足时等比例缩小
  • 某个项目设为 0 时，在空间不足时不会被压缩
  • 只有一行时 flex-shrink 生效，当存在多行（flex-wrap 生效）时不生效

初始主轴尺寸 flex-basis

设置项目在分配多余空间前占据的主轴空间。

说明：

  • 默认值为 auto，通常等同于自身内容大小或 width / height
  • 可以设定为具体长度值，例如 300px，会覆盖 width 配置

综合属性 flex

flex 是 flex-grow、flex-shrink、flex-basis 的简写。

常见写法：

  • flex: 1; 等价于 flex: 1 1 0;，常用于自适应等分布局
  • flex: auto; 等价于 1 1 auto
  • flex: none; 等价于 0 0 auto

单个项目对齐方式 align-self

允许单个项目覆盖容器的 align-items 设置。

常见取值与 align-items 基本一致：

  • auto：默认，继承 align-items
  • flex-start / flex-end / center / baseline / stretch

常用示例：水平 + 垂直居中

定位 position

position 属性概述

position 决定元素的定位方式。

常见取值：

值 说明
static 默认值，普通流
relative 相对定位
absolute 绝对定位
fixed 固定定位
sticky 粘性定位

常与以下属性配合使用：top、right、bottom、left、z-index

默认定位 static

特点：

  • 所有元素默认都是 static
  • top / left / right / bottom 无效
  • 不脱离文档流

相对定位 relative

特点：

  • 相对于自身原来的位置进行位移
  • 不脱离文档流，原位置仍然保留
  • 常用于作为绝对定位元素的参照物
  • 显示模式保持不变

绝对定位 absolute

特点：

  • 脱离文档流，不再占据原来的空间
  • 相对于最近的“已定位祖先元素”定位（position 非 static）
  • 如果没有已定位祖先，则相对于浏览器视口（通常是 body）
  • 行内块特点的显示模式

已定位元素：只要 position 不是 static，即：

  • relative
  • absolute
  • fixed
  • sticky

固定定位 fixed

特点：

  • 脱离文档流
  • 相对于浏览器视口定位
  • 页面滚动时位置不变
  • 行内块特点的显示模式

粘性定位 sticky

特点：

  • 结合了 relative 和 fixed 的特性
  • 在到达指定位置之前表现为相对定位
  • 滚动到指定位置后“吸附”在视口位置，类似固定定位
  • 必须指定 top / bottom 等偏移量之一

层级控制 z-index

z-index 用于控制定位元素在 Z 轴上的堆叠顺序（谁在上、谁在下）。

说明：

  • 数值越大，元素层级越高
  • 必须是定位元素（position 非 static）才会生效
  • 会受到“层叠上下文”的影响（如 opacity、transform 等属性也可能创建新的层叠上下文）

其他属性

内容溢出 overflow

控制内容超出盒子尺寸时的处理方式。

常见取值：

值 说明
visible 默认，内容溢出可见
hidden 裁剪溢出内容，不滚动
scroll 始终显示滚动条
auto 需要时才显示滚动条

垂直对齐方式 vertical-align

vertical-align 用于 设置元素的垂直对齐方式。

它不是用来让块级元素垂直居中的，对块级元素无效！
它主要用于：

  • 行内元素（inline / inline-block）
  • 表格单元格（table-cell）
属性值 说明
baseline 基线对齐（默认）
top 顶部对齐
middle 居中对齐
bottom 底部对齐

透明度 opacity

控制元素（包含内容、背景、边框）整体的透明度。

说明：

  • 取值范围 0 ~ 1，0 完全透明，1 完全不透明
  • opacity 小于 1 会创建新的层叠上下文

父元素配置 opacity ，子元素也会变透明。

和 background: rgba(255,0,0,0.5) 的区别：只会让背景透明，不会影响子元素。

鼠标样式 cursor

控制鼠标悬停在元素上时的指针样式。

常见取值：

值 说明
default 默认箭头
pointer 手型
text 文本选择
move 移动

过渡动画 transition

运动曲线取值 说明

linear 匀速
ease-in 慢进
ease-out 慢出
