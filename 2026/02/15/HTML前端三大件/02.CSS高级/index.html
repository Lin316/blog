层叠性

层叠性介绍：

  • 不同的属性会叠加：不同的 CSS 属性都生效。
  • 相同的属性会覆盖：高优先级的 CSS 属性会覆盖低优先级的 CSS 属性。

优先级

当多个选择器同时作用于同一个元素、并且设置了相同属性时，CSS 会计算权重最终决定哪个样式生效。

示例 1：类 vs 标签

权重对比：

  • .box：(0,0,1,0)
  • p：(0,0,0,1)
  • .box 生效

示例 2：类数量对比

权重对比：

  • .box：(0,0,1,0)
  • .box.active：(0,0,2,0)
  • .box.active 生效

示例 3：ID vs 类

权重对比：

  • #app：(0,1,0,0)
  • .box：(0,0,1,0)
  • #app 生效

示例 4：元素 + 类 vs 单个类

CSS Sprites 雪碧图

CSS Sprites 介绍

CSS Sprites（雪碧图）是一种性能优化技术：把多个小图片合并成一张大图片，然后通过 background-position 精确显示其中一部分。

为什么要用 CSS Sprites：

  1. 传统问题
    • 页面有 20 个小图标，每个图标一个图片文件
    • 浏览器需发起 20 次 HTTP 请求 → 请求多 → 页面加载慢
  2. 使用雪碧图
    • 将 20 个图标合成一张图，仅发 1 次请求
    • 显著减少请求数，性能更好

CSS Sprites 优点：

优点 说明
减少请求次数 合并小图，降低 HTTP 请求数
性能优化 加快页面加载速度（尤其在 HTTP/1.1 下）
统一管理 图标集中在一张图，便于团队协作与维护

CSS Sprites 缺点：

缺点 说明
修改麻烦 改一个图标需要重新合成与定位
不够灵活 更适合小图标，不适合大图与照片
体积可能大 合图过大可能影响首屏加载

字体图标

字体图标介绍

字体图标本质上是“字体文件”，只不过里面存放的不是字母，而是各种图标（如搜索、购物车、删除等）。

它的核心思想是：把图标当成“文字”来使用。

字体图标的特点：

优点 说明
体积小 一个字体文件包含多个图标
清晰不失真 本质是矢量图，可无限放大
颜色可控 通过 color 修改
大小可控 通过 font-size 控制
加载快 只需请求一个字体文件
