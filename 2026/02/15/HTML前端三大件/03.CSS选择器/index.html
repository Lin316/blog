标签选择器

标签选择器：使用标签名作为选择器，选中同名标签设置相同的样式，例如：p、div、a、img 等标签。

特点：

交集选择器

交集选择器：选中同时满足多个条件的元素。

基本写法：选择器1选择器2（中间没有空格）。

并集选择器

并集选择器：选中多个不同选择器对应的所有元素，多个选择器之间使用英文逗号分隔。

后代选择器

  • 选中 div 里面的所有 p（不限层级）。

子代选择器

  • 只选中 div 的直接子元素 p。

相邻兄弟选择器

  • 选中紧跟在 h1 后面的第一个同级 p。

通用兄弟选择器

  • 选中 h1 后面所有同级的 p。

属性选择器

属性选择器：根据元素是否存在某个属性，或属性值是否满足条件来选中元素。

存在某个属性

  • 选中所有带有 type 属性的 input。

指定属性值

  • 选中所有 type=”text” 的 input。

模糊匹配

写法说明：

  • [attr^=val] 以 val 开头
  • [attr$=val] 以 val 结尾
  • [attr*=val] 包含 val

示例：

  • 选中所有 href 以 https 开头的链接。

:nth-child() 和 :nth-of-type() 的区别：

  • li:nth-child(2)：获取父标签中所有类型的子标签集合，从 1 开始编号，选中序号为 2 的子标签。
  • :nth-of-type(2)：获取父标签中指定类型的子标签集合，从 1 开始编号，选中序号为 2 的子标签。

注：:nth-child() 支持 an + b 数学表达式。

伪元素选择器

伪元素：

  • 不是 HTML 里真实存在的元素。
  • 是 CSS “创造”出来的虚拟元素。
  • 必须用 :: 双冒号（CSS3标准写法）。

常见伪元素：

  • p::before：被选中的标签添加一个 span 子元素，并成为父标签的第一个子元素。
  • p::after：被选中的标签添加一个 span 子元素，并成为父标签的最后一个子元素。
  • p::first-letter：选中标签里面的第一个字母，即使这个字母在嵌套多层的标签中也能被选中。
  • p::first-line：选中标签里面的第一行文本，即使这行文本在嵌套多层的标签中也能被选中。
  • input::placeholder：选中表单输入框中的提示文本。
