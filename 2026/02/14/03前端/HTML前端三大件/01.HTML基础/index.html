HTML 介绍

HTML 介绍： 用来构建网页结构的标准标记语言，通过“标签（Tag）”来描述网页内容的结构。

HTML 基本结构：

标签 作用
<!DOCTYPE html> 声明使用 HTML5
<html> 整个网页的根元素
<head> 页面元信息（标题、编码、样式等）
<title> 浏览器标签页标题
<body> 页面可见内容

<h> 标题标签

用法

特点

  1. 字体加粗。
  2. 上下有外边距。

注意事项

  1. 合理使用标题层级有助于搜索引擎排名。
  2. 一个页面只能有一个 h1 标签。

<p> 段落标签

用法

特点

  1. 块级元素，独占一行，默认带上下外边距（通常 16px 左右）。
  2. 表示“一个语义完整的文本段落”，对搜索引擎、屏幕阅读器、文档结构都有意义。

注意事项

  1. 不能跟 <p> 嵌套的标签：<div>、<h1>~<h6>、<section>、<article>。
  2. 可以跟 <p> 嵌套的标签：文本、<span>、<a>、<strong>、<em>、<img> 等。

<br> 换行标签

用法

特点

  1. 不需要结束标签。
  2. 属于行内元素。
  3. 不带上下间距。

注意事项

  1. 不要使用多个 <br> 做布局间距，如 <br><br><br>。

<hr> 水平线标签

用法

特点

  1. 可以使用 css 改成渐变线、虚线、带文字分割线、双线等。

文本格式化标签

用法

<strong>：给字体加粗。

<em>：让字体倾斜。

<del>：给字体加上删除线。

<ins>：给字体加上下滑线（用 css 实现下滑线，不用这个了）。

<sup>：上标，如：x²。

<sub>：下标，如：H₂O。

注意事项

  1. 使用 <strong>，而不是 <b>。
  2. 使用 <em>，而不是 <i>。
  3. 使用 <del>，而不是 <s>。

<img> 图像标签

用法

属性 作用
src 图片路径（必填），用相对路径。
alt 替代文本（必填），图片加载失败时显示的文字。
title 鼠标悬停时显示的文字。

特点

  1. 默认行内元素。
  2. 可以用懒加载提高性能（只有滚动到可视区域才加载）。

注意事项

  1. 绝对路径：url 以 / 开头或以 win 的盘符开头。

  2. 相对路径：url 以 ./ 或 文件夹名/开头。

  3. 路径统一用 / 符号。

<a> 超链接标签

用法

跳转外部网站

属性 作用
target 超链接的打开方式：_self当前页面打开（默认）、_blank 新窗口打开。
title 鼠标悬停时显示的文字。
download 下载文件。

特点

  • 跳转到页面内部指定位置。
  • 跳转到系统其它页面。
  • 跳转到外部网站，例如百度页面。
  • 发送邮件。
  • 拨打电话（移动端）。

<audio> 音频标签

用法

属性 作用
controls 显示控制面板，如：播放、暂停、进度条、音量控制。
autoplay 自动播放。
loop 循环播放。

特点

注意事项

  1. 支持的音频格式：mp3、wav、ogg。
  2. autoplay 不生效原因：可能是浏览器防止骚扰。

<video> 视频标签

用法

属性 作用
controls 显示控制面板，如：播放、暂停、进度条、音量控制。
autoplay 自动播放。
loop 循环播放。
muted 静音。

注意事项

  1. 视频支持的格式：mp4、webm、ogg。
  2. 现代浏览器限制自动播放，必须将autoplay muted配合使用。

列表标签

用法

有三种列表：无序列表、有序列表、定义列表。

注意事项

  1. 无序列表 ul 标签只能嵌套 li 标签。
  2. 有序列表 ol 标签只能嵌套 li 标签。
  3. 定义列表 dl 标签只能嵌套 dt 标签 和 dd 标签。

<table> 表格标签

用法

合并单元格：

  • 跨行合并：保留最上单元格，td 标签添加 rowspan 属性。
  • 跨列合并：保留最左单元格，td 标签添加 colspan 属性。

注意事项

  1. 表格默认没有边框线。

<form> 表单标签

用法

<form> 标签：收集用户输入的信息并提交到服务器。

<input> 标签：用户可以在其中输入数据。

<select> 标签：下拉列表，提供多选操作。

<textarea> 标签：文本域，用于多行文本。

<label> 标签：使用 for 属性绑定表单标签的 id 属性，点击文字时，会自动聚焦或选中对应的输入控件。

属性

form 标签的属性 作用说明
action 表单提交的服务器地址
method 提交方式（GET / POST）
target 提交后在哪打开（_self / _blank）
enctype 表单数据编码方式（文件上传时必须设置）
autocomplete 是否自动填充
input 标签的 type 属性取值 说明
text 文本框
password 密码框
radio 单选按钮
checkbox 复选框
file 文件上传
submit 提交按钮
reset 重置按钮
button 普通按钮
email 邮箱
number 数字
date 日期
hidden 隐藏域

input 标签 placeholder 属性：当 input 或 textarea 为空时，会显示一段灰色提示文字，提示用户该输入什么内容。

注意事项

  1. 只有带 name 的表单元素才会被提交到服务器。
  2. 上传文件必须设置 form 标签的 enctype 属性。
  3. 在单选按钮中，同一组选项的 name 属性值必须相同。
