Emmet 介绍

Emmet 是一个前端开发效率工具，可以用简写语法快速生成 HTML 和 CSS 代码。

用类似 CSS 选择器的语法，按 Tab 键自动展开为完整代码。

HTML 基础结构语法

写法 含义 展开结果
div 标签 <div></div>
div.box class <div class="box"></div>
div#app id <div id="app"></div>
div.box#app class + id <div class="box" id="app"></div>
ul>li 子元素 <ul><li></li></ul>
h1+p 兄弟元素 <h1></h1><p></p>
div>ul>li^p 返回上级 <div><ul><li></li></ul><p></p></div>
li*3 生成多个 生成 3 个 <li></li>
li.item$*3 自动编号 class="item1"、item2、item3
li.item$$$*3 补零编号 class="item001"、item002、item003
p{Hello} 文本内容 <p>Hello</p>

提示：$ 表示自动编号，$$$ 会自动补零到 3 位数。

HTML 组合写法

写法 说明 生成结构说明
ul>li.item$*3 列表 + 编号 3 个带 class 的 li，item1/2/3
div.container>ul.list>li*3 嵌套结构 常见页面骨架结构
a[href=#] 属性写法 <a href="#"></a>
img[src=1.jpg] 设置属性 <img src="1.jpg">
input:text 文本输入框 <input type="text">
input:password 密码输入框 <input type="password">
! HTML5 模板 生成完整 HTML5 文档结构
html:5 HTML5 模板 同上，生成标准 HTML5 页面骨架

CSS 属性写法

简写 展开
w100 width: 100px;
h200 height: 200px;
m10 margin: 10px;
p20 padding: 20px;
fs16 font-size: 16px;
fw700 font-weight: 700;
lh30 line-height: 30px;
bgc background-color: ;
bg background: ;
c#fff color: #fff;
bd1 border: 1px solid #000;

提示：实际书写时，光标会停在分号前，方便继续编辑数值或颜色。

CSS Flex 布局写法

简写 展开
d:f display: flex;
fxd:r flex-direction: row;
fxd:c flex-direction: column;
jc:c justify-content: center;
jc:sb justify-content: space-between;
ai:c align-items: center;
fw:w flex-wrap: wrap;

CSS 背景图写法

简写 展开
bgi background-image: url();
bgr background-repeat: no-repeat;

CSS 定位写法

简写 展开
pos:r position: relative;
pos:a position: absolute;
pos:f position: fixed;
t0 top: 0;
l0 left: 0;
r0 right: 0;
b0 bottom: 0;
