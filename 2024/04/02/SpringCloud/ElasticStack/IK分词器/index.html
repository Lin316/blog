安装 IK 分词器

  1. IK 分词器官网下载地址，选择跟 es 版本兼容的 IK 版本。
  2. 在 es 的 plugins 目录中创建 ik 目录，把下载的 zip 包放到 ik 目录中。
  3. 在 linux 中使用 unzip 包名.zip 解压 ik 压缩包。
  4. 重启 es。

IK 分词模式

IK 分词器有两种分词模式：

  • ik_smart：粗粒度。
  • ik_max_word：细粒度。

IK 分词器的配置文件

config 目录

  • main.dic：ik 内置的中文词库。
  • preposition.dic: 介词。
  • quantifier.dic：量词。
  • suffix.dic：中文后缀。
  • surname.dic：姓氏
  • stopword.dic：英文停用词
  • IKAnalyzer.cfg.xml：用来配置扩展词库和停用词。
