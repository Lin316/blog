session 会话对象

作用：session 会话对象能够跨请求保持某些参数。它也会在同一个 Session 实例发出的所有请求之间保持 cookie。session 会话对象具有 Requests API 的所有方法。

创建 Request 对象

request 发送请求的原理

当使用 requests 库的 get/post 方法发送请求时，requests 在内部构造了一个 Request 对象，并给这个对象赋予了各种参数，包括 url、headers、data 等等。然后直接把这个 Request 对象发送出去，请求成功后会再得到一个 Response 对象。
