高阶函数

map

map() 函数接收两个参数：一个是函数，一个是 Iterable 对象。 map 将传入的函数依次作用到序列的每个元素，并把结果作为新的 Iterator 返回。

传入函数的特点：接收一个参数，返回一个值。

reduce

传入函数的特点：接收两个参数，返回一个值。

filter

filter() 函数接收一个函数和一个序列。filter() 把传入的函数依次作用于每个元素，然后根据返回值 True/False 决定是否保留该元素。

传入函数的特点：接收一个参数，返回 True/False 。

sorted

sorted() 函数接收三个参数：序列，key，reverse。其中 key 是个默认参数，接收一个映射函数。reverse 也是个默认参数，传入 True/False 表示是否反转排序后的序列。

sorted() 函数默认按从小到大排序。

传入函数的特点：接收一个参数，返回 int 值。
